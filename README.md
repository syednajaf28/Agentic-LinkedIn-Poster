# 🚀 Autonomous LinkedIn AI Agent

## 📌 Project Overview
I successfully orchestrated an **Agentic AI** workflow using Claude through the terminal to achieve a high-level goal. In this project, the AI operated autonomously—making real-time decisions and executing tasks without manual coding.

## 🛠️ How it Works (The Process)
Using a combination of **Python** and **Playwright** in the backend, the AI Agent performed the following actions:
1. **Autonomous Browser Management:** Handled web navigation and element interaction independently.
2. **Automated LinkedIn Integration:** Logged into LinkedIn and navigated the interface securely.
3. **Content Curation:** Fetched trending news, generated concise summaries, and published posts on the platform.

## 💡 Key Skills Demonstrated
This project highlights my proficiency in:
* **Agentic AI Orchestration:** Managing and directing AI agents via terminal interfaces.
* **Advanced Prompt Engineering:** Crafting complex instructions to guide AI through multi-step workflows.
* **Browser Automation:** Leveraging AI to control web interfaces and automate repetitive online tasks.
